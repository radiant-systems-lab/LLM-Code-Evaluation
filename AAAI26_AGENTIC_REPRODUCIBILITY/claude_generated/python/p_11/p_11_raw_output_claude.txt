## Quick Start Guide

### Step 1: Create the Files

Copy each code block above into separate files with the names indicated:

1. `email_automation.py` - Main script
2. `requirements.txt` - Dependencies
3. `email_template.html` - HTML template
4. `config.py` - Configuration
5. `.env.example` - Environment template
6. `example_usage.py` - Examples
7. `README.md` - Documentation
8. `.gitignore` - Git ignore

### Step 2: Install Dependencies (Optional)

```bash
pip install -r requirements.txt
```

Note: The script uses only Python standard library. `python-dotenv` is optional for easier configuration management.

### Step 3: Configure Gmail App Password

1. Enable 2-Step Verification in your Google Account
2. Go to Security → App passwords
3. Create a new app password for "Mail"
4. Copy the 16-character password

### Step 4: Set Up Environment

```bash
cp .env.example .env
# Edit .env with your credentials
```

### Step 5: Run Examples

```bash
# Run the example script
python example_usage.py

# Or use directly in your code
python email_automation.py
```
