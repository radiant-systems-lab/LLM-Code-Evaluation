## Quick Start:

```bash

pip install -r requirements.txt

# Generate a QR code
python qr_app.py generate "https://example.com" my_qr

# Read a QR code
python qr_app.py read qr_codes/my_qr.png

# Run all examples
python example_usage.py
```
