# Real-Time Data Visualization Dashboard

This project is a real-time data visualization dashboard built with Express, Socket.io, and Chart.js.

## Requirements

- Node.js
- npm

## Installation

1.  Install the dependencies:

    ```bash
    npm install
    ```

## Usage

1.  Start the server:

    ```bash
    npm start
    ```

2.  Open your browser and go to `http://localhost:3000`.

3.  The charts will update in real-time with data from the server.
