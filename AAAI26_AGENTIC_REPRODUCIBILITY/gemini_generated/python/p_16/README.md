# Collaborative Filtering Recommendation System

This project demonstrates how to build a recommendation system using user-based and item-based collaborative filtering. It uses the `scikit-surprise` library, which is designed for building and analyzing recommendation systems.

## Features

- **Built-in Dataset**: Uses the classic MovieLens 100k dataset, which is automatically downloaded by the library, making the script fully self-contained.
- **User-Based Filtering**: Implements a k-Nearest Neighbors model configured to find similar users and recommend items they liked.
- **Item-Based Filtering**: Implements a second k-NN model configured to find similar items and recommend them based on a user's rating history.
- **Top-N Recommendations**: For a sample user, the script calculates and prints the top 10 movie recommendations generated by each of the two models.

## How It Works

1.  **User-Based Collaborative Filtering**: This method finds users who have similar rating patterns to the target user. It then recommends items that these similar users have rated highly but that the target user has not yet seen.
2.  **Item-Based Collaborative Filtering**: This method builds a similarity matrix between items based on user ratings. It recommends items that are similar to those that the target user has already rated highly.

## Usage

1.  **Set up a virtual environment:**
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows, use `venv\Scripts\activate`
    ```

2.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the script:**
    ```bash
    python recommender.py
    ```
