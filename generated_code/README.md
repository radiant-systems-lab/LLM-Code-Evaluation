# Generated Code

300 AI-generated projects: 100 prompts x 3 agents.

## Agents

| Directory | Agent | Description |
|-----------|-------|-------------|
| `claude/` | Claude Code | Anthropic's CLI coding agent |
| `codex/` | GitHub Copilot | OpenAI Codex via VS Code |
| `gemini/` | Gemini Code Assist | Google's coding agent |

## Structure

Each agent directory contains three language subdirectories:

```
{agent}/
├── python/       # p_1 to p_40 (40 projects)
│   └── p_X/
│       ├── *.py                    # Generated script
│       ├── requirements.txt        # Declared Python dependencies
│       └── package.json / pom.xml  # (for JS/Java projects)
├── javascript/   # p_41 to p_75 (35 projects)
│   └── p_X/
│       ├── *.js                    # Generated script
│       └── package.json            # Declared npm dependencies
└── java/         # p_76 to p_100 (25 projects)
    └── p_X/
        ├── *.java / src/           # Generated source
        └── pom.xml                 # Maven dependencies
```

## Notes

- Code is preserved exactly as generated by each agent (no manual edits to source)
- Dependency files (requirements.txt, package.json, pom.xml) are as declared by the agent
- Some projects include metadata files documenting generation context
