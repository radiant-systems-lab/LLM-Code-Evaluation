## Key Features Implemented

1. **Async/Concurrent** - Uses `asyncio.gather()` for parallel crawling
2. **Robots.txt Compliance** - Fetches, parses, and respects robots.txt per domain
3. **Rate Limiting** - Tracks last access time per domain and enforces delays
4. **Semaphore Control** - Limits max concurrent requests system-wide
5. **File Saving** - Async file writes with URL and timestamp metadata
6. **Error Handling** - Graceful handling of timeouts, HTTP errors, etc.
