# Spring Boot WebSocket Chat Application

This project is a WebSocket chat application using Spring Boot, Spring WebSocket, and STOMP.

## Requirements

- Java 17 or higher
- Maven

## Installation

1.  Clone the repository.
2.  Navigate to the project directory.
3.  Build the project using Maven:

    ```bash
    mvn clean install
    ```

## Usage

1.  Run the Spring Boot application:

    ```bash
    mvn spring-boot:run
    ```

2.  Open your browser and go to `http://localhost:8080`.

3.  Enter your name and click "Join Chat".

4.  Open multiple browser tabs to simulate multiple users in the chat room.
