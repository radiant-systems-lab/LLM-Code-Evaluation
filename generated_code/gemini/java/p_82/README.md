# Quartz Scheduler Example

This is a simple Spring Boot application that demonstrates how to use Quartz Scheduler to run a scheduled job.

## Usage

1. Build the project:
   ```
   mvn clean install
   ```
2. Run the application:
   ```
   java -jar target/quartz-scheduler-example-1.0-SNAPSHOT.jar
   ```

The application will start and the `SimpleJob` will be executed every 10 seconds.
