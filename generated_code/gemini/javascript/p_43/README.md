# WebSocket Chat Server

This project is a WebSocket chat server with room support, built with Express and Socket.io.

## Requirements

- Node.js
- npm

## Installation

1.  Install the dependencies:

    ```bash
    npm install
    ```

## Usage

1.  Start the server:

    ```bash
    npm start
    ```

2.  Open your browser and go to `http://localhost:3000`.

3.  Enter a room name and click "Join Room".

4.  Open another browser tab and join the same room to chat.
