# PDF Generation Service

This project is a PDF generation service from HTML, built with Express and Puppeteer.

## Requirements

- Node.js
- npm

## Installation

1.  Install the dependencies:

    ```bash
    npm install
    ```

## Usage

1.  Start the server:

    ```bash
    npm start
    ```

2.  Open your browser and go to `http://localhost:3000/generate-pdf` to download the generated PDF.
