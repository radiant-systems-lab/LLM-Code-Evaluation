# Web Scraper

This project is a web scraper that uses Puppeteer to scrape quotes from a dynamic website and saves them to a JSON file.

## Requirements

- Node.js
- npm

## Installation

1.  Install the dependencies:

    ```bash
    npm install
    ```

## Usage

1.  Run the scraper:

    ```bash
    npm start
    ```

2.  The scraper will scrape quotes from `http://quotes.toscrape.com/js/` and save them to a `quotes.json` file in the project directory.
