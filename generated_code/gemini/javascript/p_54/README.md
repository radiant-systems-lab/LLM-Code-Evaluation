# File Watcher

This project is a file watcher that triggers actions on changes, built with the Chokidar library.

## Requirements

- Node.js
- npm

## Installation

1.  Install the dependencies:

    ```bash
    npm install
    ```

## Usage

1.  Start the watcher:

    ```bash
    npm start
    ```

2.  The watcher will monitor the `watched-dir` directory for changes. Any file or directory changes will be logged to the console.
