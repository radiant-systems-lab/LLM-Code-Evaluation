# Real-Time Notification System

This project is a real-time notification system built with Express, Socket.io, and MongoDB.

## Requirements

- Node.js
- npm
- MongoDB

## Installation

1.  Install the dependencies:

    ```bash
    npm install
    ```

2.  Make sure you have a MongoDB server running.

## Usage

1.  Start the server:

    ```bash
    npm start
    ```

2.  Open your browser and go to `http://localhost:3000`.

3.  You will see real-time notifications appearing on the page. You can mark notifications as read.
