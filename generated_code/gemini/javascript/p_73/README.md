# Markdown Preview Server

This project is a markdown preview server with live reload, built with Express, Marked, Chokidar, and WebSockets.

## Requirements

- Node.js
- npm

## Installation

1.  Install the dependencies:

    ```bash
    npm install
    ```

## Usage

1.  Start the server:

    ```bash
    npm start
    ```

2.  Open your browser and go to `http://localhost:3000`.

3.  Edit the `document.md` file, and the browser will automatically refresh to show the changes.
