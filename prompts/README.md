# Prompts

100 standardized task prompts used to generate code from all three AI coding agents.

## Distribution

| Language | Range | Count | Focus Areas |
|----------|-------|-------|-------------|
| Python | p_1 - p_40 | 40 | Web scraping, ML/AI, data processing, APIs, NLP, encryption, monitoring |
| JavaScript | p_41 - p_75 | 35 | Express REST APIs, WebSocket, React, OAuth, file processing, job queues |
| Java | p_76 - p_100 | 25 | Spring Boot, Kafka, JPA, batch processing, caching, rate limiting |

## Format

Each prompt file contains a task description requesting complete, executable code with all necessary dependencies. Prompts are identical across all three agents to ensure fair comparison.
